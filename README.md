# HR-Analytics-PowerBI
This project analyzes employee retention at a fictional company using the IBM HR Analytics dataset. The goal was to build an interactive dashboard that helps HR managers identify why employees are leaving and which departments are most affected.

🚀 Key Features
Total Headcount Tracking: Real-time calculation of total employees.

Attrition Breakdown: Visual representation of "Stayed" vs. "Left" using interactive Pie Charts.

Departmental Analysis: A slicer-driven interface that allows users to filter the entire dashboard by Department (Sales, R&D, and HR).

Salary Insights: Analysis of Average Monthly Income to see if pay levels correlate with employee turnover.

🛠️ Tools & Technologies
Power BI Desktop: For data visualization and dashboard building.

Power Query: For data cleaning, renaming columns, and changing data types.

Data Modeling: Connecting employee metrics to organizational categories.

📊 Business Insights
Retention vs. Pay: The data indicates that employees who leave typically have a lower average monthly income, suggesting that compensation is a primary driver for attrition.

High-Risk Areas: The Sales department experiences higher turnover rates compared to Research & Development.

Data-Driven Decisions: With the interactive slicer, HR can now pinpoint specific departments that need retention bonuses or better engagement strategies.
